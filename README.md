## 🎮 What is Sift?

Sift uses AI to analyze your gameplay recordings and automatically find exciting moments — kills, clutch plays, and highlight-worthy clips.

**Stop scrubbing through hours of footage.** Let Sift find the gold.

<br>

## ⚡ Features

- **🤖 AI-Powered Detection** — Automatically detects action moments using advanced audio analysis
- **✂️ Smart Trimming** — Fine-tune clip boundaries with frame-accurate controls  
- **🎬 Quick Export** — Export individual clips or batch export all moments
- **🎨 Beautiful UI** — Clean, modern interface designed for content creators
- **🔄 Auto Updates** — Stay up to date with one-click updates

<br>

## 📥 Installation

1. **Download** [Sift-Setup.exe](https://github.com/JashVaidya/sift-release/releases/latest/download/Sift_0.1.0_x64-setup.exe)
2. **Run** the installer
3. **Launch** Sift from the Start Menu

> **Note:** Windows SmartScreen may show a warning since the app isn't signed with a paid certificate. Click "More info" → "Run anyway" to proceed. This is normal for indie software.
